Smart Portfolio Tracker

This application is designed to help you easily manage and track your stock market investments. It provides a centralized view of your portfolio, allowing you to monitor performance, analyze holdings, and stay informed about market trends.


Key Features

  • Real-time stock price updates
  • Portfolio performance tracking (gains/losses, total value)
  • Detailed holdings information (quantity, purchase price, current value).
  • User-friendly interface for easy navigation and data visualization.

    • Technology Stack

  • Java with Spring Boot framework
  • PFrontend : Thymeleaf (for a dynamic and interactive user experience.)
  • Database used : MySQL (for secure and persistent data storage)
